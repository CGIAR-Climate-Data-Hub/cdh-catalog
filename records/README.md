# Metadata Records

This directory contains the source metadata records for the Climate Data Hub
catalog. This is where datasets and other resources need to be submitted to be
integrated into the Climate Data Hub.

Each subdirectory represents a cataloged resource, such as a dataset, tool,
report, model, notebook, API, or service. These YAML files are the
human-authored files used to generate the published STAC and OGC API Records
catalogs and entries.

## Structure

Each directory contains the following files:

```text
records/
  dataset1/
    record.yaml
    additional_metadata.md
    classes.json
  dataset2/
    record.yaml
  dataset3/
    record.yaml
```
